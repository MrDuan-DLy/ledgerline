# Changelog

## [1.0.0] - 2026-03-07

### Added
- Go backend with Chi router, sqlx, and SQLite (modernc.org/sqlite)
- Bank statement PDF import with Gemini AI extraction
- CSV import with auto-deduplication
- Transaction management with search, filter, and pagination
- Rule-based auto-classification with priority ordering
- Receipt image upload with Gemini OCR extraction
- Smart receipt-to-transaction matching (fuzzy date/amount/merchant)
- Merchant management with pattern learning and fuzzy matching
- Budget tracking with monthly spend monitoring
- Interactive dashboard with spend charts and spending pace
- Docker deployment with multi-stage build
- GitHub Actions CI/CD pipeline
- React frontend with Vite and TypeScript
- AppConfig context for currency/locale customization
